### 2024년에 만들어진 BSSM 전공동아리입니다.

<br>

# ✨추구하는 가치
1. 창의성
   - 새로운 아이디어와 혁신적인 생각을 장려합니다.
2. 책임감
   - 자신의 역할과 업무에 책임을 다합니다.
   - 각자의 역할을 명확히 하고, 결과에 대해 책임질 수 있어야해요.
3. 학습
   - 지속적인 학습과 자기 개발을 추구합니다.

<br>

# 🫂팀문화
1. 자기계발
   - 스스로 찾아서 공부하고 일해요!
   - 새로운 기술을 무서워하지 않아요.
2. 목표지향
   - 스스로 목표를 정하고 열정으로 달성하기 위해 노력해요!
